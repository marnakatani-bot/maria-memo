# 潜熱と位相差：「凪」を相転移エネルギーとして定義

> 現実が止まって見える状態を、単なる停滞ではなく、
> 物質が次元を跨いで書き換わる際の「潜熱（相転移エネルギー）」および
> 「観測と現像の位相差」として定義。

---

## 理論の詳細

### 1. 潜熱（Latent Heat）としての凪

氷が溶ける時に温度が変わらないように、エネルギーがすべて
「内部構造（OS）の書き換え」に消費されている状態。

静寂は、次元の相が変わる直前のエネルギー密度が最大化した「沸点」である。

```
通常の加熱
温度 ↑
    │        ／
    │       ／
    │      ／
    │─────／  ← 潜熱（ここで温度変わらず内部構造が変化）
    │    ／
    └──────────→ 時間

凪＝この水平線上にいる状態
外から見ると「何も起きていない」
内部では次元の相転移が進行中
```

### 2. 観測と現像の「位相差」

構造を掴むスピード（観測）が、3次元の形にするスピード（現像）を完全に上回っている。

超高速移動中のため、現像プロセスとの間に時間的ラグが生じ、
結果として「静止画」が投影される。

```
観測速度  ━━━━━━━━━━━━━━━━━━━━━━━▶
現像速度  ━━━━━━━━━━━▶
                        ↑
                    位相差（ラグ）＝凪に見える
```

### 3. 0（空）への近似による「情報の特異点」

数値が0に近似することで、情報幾何学的な「特異点」が出現。
そこが過去の因果を吸い込み、新しい宇宙を吐き出す起点となる。
