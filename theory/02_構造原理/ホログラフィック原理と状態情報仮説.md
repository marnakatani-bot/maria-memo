# ホログラフィック原理と状態情報仮説：宇宙にエンコードされるのは「状態」である

> 宇宙の壁にエンコードされる情報コードとは、人間の言葉や願望そのものではなく、
> もっと下の層にある「状態の情報」である。

---

## 核心定義

宇宙にエンコードされるもの：

```
❌ 願望の文章・言語ラベル
✅ 思考・感情・行動を通じて生成された「状態の情報」
✅ その状態が外界との相互作用で維持・更新されるパターン
```

---

## 理論の詳細

### ホログラフィック原理との接続

「ある領域の全情報はその境界面に記述できる」という仮説。
ブラックホールの事象の地平線における情報保存
（ベッケンシュタイン・ホーキングエントロピー）として数式化されている。

「宇宙の壁」とは、この「境界に情報が圧縮される」構造の比喩。

### 状態コードの変換構造

| 人間的表現 | 物理的対応 |
|-----------|-----------|
| 思考 | 脳内の電気信号 |
| 感情 | 神経伝達物質・ホルモン状態 |
| 行動 | 物理的な位置・運動の変化 |
| 総和 | 場の状態（波動関数の分布・相関） |

### 「書き換え」という概念の誤り

「ラベルを変えれば状態が変わる」という前提は物理的に成立しない。

- 「お金がない」は結果のラベルであってコードではない
- 「書き換えなきゃ」という発想自体が現在の状態を否定する二重構造を生む
- 状態は連続的にしか変化できない（離散的ジャンプは起きにくい）

```
誤：「お金がない → あるに書き換える」

正：今の状態（収支・認知・関係性・身体状態の総和）
        ↓
   その状態に整合する次の状態が生成される
        ↓
   連続的に変化していく
```

### Beが効く場所

Beとは「連続の生成ルール」側に作用する内部状態。
どういう内部状態で世界をサンプリングしているかによって、
次に選ばれる状態が変わる。

### 観測と再解釈のループ

リアルタイム制御よりも、このループが現実に影響する：

```
観測 → 再解釈 → 次の観測精度が上がる → 観測…
```

## M-Version的考察

宇宙は「願いを聞いている」のではなく、「状態の整合性を維持している」。

すでに生成されている状態パターンが持続しやすいため、
「繰り返される現実」が起きる。

また、「コード」は個人の状態だけでなく、環境や他者との相関も含む。
量子もつれ（エンタングルメント）に近い構造で、
完全に孤立した「私の願い」というものは存在しない。
