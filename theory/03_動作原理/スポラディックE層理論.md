# スポラディックE層（Eスポ）理論：次元上昇後の沈黙と異常シンクロの物理定義

> 5Dから7D以上へ次元上昇したことで起きた「集客の停止」と、
> 特定条件下で起きた「りんね氏との異常なシンクロ」を、
> 電波工学の「スポラディックE層（Eスポ）」として定義。

---

## 理論の詳細

### 1. 高周波の宇宙透過（7D以降の沈黙）

まり亜さんの周波数が高次化したため、通常の3D電離層を突き抜けて宇宙へ抜けてしまう。
結果、3D住人には「放送（レンタルまり亜）」が受信できず、人が来なくなる。

```
7D以上の周波数
        ↓
3D電離層を突き抜ける
        ↓
宇宙へ抜ける（3D住人には受信不可）
        ↓
集客の停止・沈黙
```

### 2. Eスポによる異常伝播（瞬間のシンクロ）

| 要素 | 内容 |
|------|------|
| りんね氏の極限 | 3Dドラマの極限（親の死） |
| まり亜さんの極限 | 物理的極限（80円） |
| 結果 | 一時的な「高密度電子層（Eスポ）」を形成 |

本来繋がらないはずの周波数が反射し、事故的にシンクロが発生した。

### 3. 物理的フェードアウト（音信不通の正体）

- Eスポは突発的で短寿命
- 気象条件（磁場・エネルギー）が変われば反射板は消滅し、通信は物理的に途絶する
- 音信不通は「通信路の消滅」であり、個人の感情の問題ではない

---

## 物理的検証

**スポラディックE層（Es）**:
通常は電波を反射しない高度に、突如として発生する強い電離層。
VHF帯などの高周波を異常反射させる。

**ヘテロダイン混信**:
異なる周波数が無理やり重なることで生じる「うなり（不協和音）」。

## 全体構造図

```
通常状態
まり亜（7D+）───→ 3D電離層を透過 ───→ 宇宙へ（受信不可）

Eスポ発生時
りんね（3D極限）
        ＋                → 高密度電子層（Eスポ）形成
まり亜（80円極限）               ↓
                        異常反射・事故的シンクロ発生

Eスポ消滅後
気象条件変化 → 反射板消滅 → 通信途絶（音信不通）
```

## M-Version的考察

カエルが降る（あり得ない縁が繋がる）のは、次元の気象異常に過ぎない。
晴れれば（次元が安定すれば）ノイズは消える。

13へ進むためには、この「混信」に執着せず、
通信途絶を「正常な次元隔離」として受け入れる必要がある。
